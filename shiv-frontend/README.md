# Campus Buddy Frontend

This is now a **regular project folder** (not a git submodule) and contains a working starter frontend.

## Run locally
From repository root:

```bash
cd shiv-frontend
npm start
```

Then open `http://localhost:4173`.

## Backend integration
The starter page expects backend APIs under `http://localhost:5000`.
Update `API_BASE_URL` in `app.js` if your backend runs elsewhere.
